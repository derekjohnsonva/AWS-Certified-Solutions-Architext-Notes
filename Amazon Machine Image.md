A way to create [[Elastic Compute Cloud|EC2]] instances with certain configurations
There is a marketplace that includes commercial software
AMIs are Regional (Only can be used in one region)
You can create an AMI from an existing EC2 instance
An AMI does not contain any data, it just points to data in an [[EBS]]
AMI Baking = creating an AMI from a configured instance + application
An AMI can not be edited
Data can be copied between regions
The default permission is only available to your account.
